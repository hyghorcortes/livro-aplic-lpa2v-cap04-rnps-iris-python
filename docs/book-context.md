# Contexto no livro

Este repositório foi preparado como material complementar do **Capítulo 4 — RNPs (Redes Neurais Paraconsistentes)** do livro **Aplicações de LPA2v**.

O exemplo usa a base Iris para demonstrar, de forma prática, como arquiteturas neurais convencionais e paraconsistentes podem ser estruturadas, executadas e comparadas em Python.

Links que podem ser preenchidos posteriormente no apêndice do livro:

- repositório principal do exemplo;
- scripts das variantes NN, CNAPa, CNAPap e CNAPCa;
- arquivos de resultados em CSV e JSON;
- figuras geradas pelos experimentos.
