# Código de Conduta

Este projeto adota um padrão simples de convivência:

- respeito mútuo;
- comunicação técnica clara;
- foco na colaboração;
- abertura para discussão acadêmica e engenharia responsável.

Comportamentos inadequados incluem ataques pessoais, linguagem ofensiva, assédio, intimidação ou desqualificação gratuita do trabalho alheio.

Os mantenedores podem remover contribuições, comentários ou interações que prejudiquem o ambiente colaborativo.
